== README

 
This application is recommended to be used with

* Ruby version 2.1.0

* Rails version 4.0.4

* Sqlite 3

* Take the follow steps to setup and run the app

  * Run "bundle install" to install the required gems

  * Run "rake db:migrate" to create database tables(If not using sqlite, you may need to run db:create first)

  * Run "rails s" to run the web application
